# Deployment Guide

## Recommended Hosting

- GitHub Pages
- Netlify
- Azure Static Web Apps

## Custom Domain Steps

1. Buy or use an existing domain.
2. Point DNS to your hosting provider.
3. Add the domain in hosting settings.
4. Enable HTTPS.

## Publish Checklist

- Confirm all image and video paths exist.
- Keep `manifest.webmanifest` and `service-worker.js` in the root.
- Test mobile navigation, media, PDF export, and offline loading.
- Verify `privacy-policy.html` and `executive-summary.html` open correctly.
